# Ак барс банк: обучение модели для скринингового анализа
Тестовое задание для прохождения в Ак барс банк на вакансию риск-менеджера (ML моделирование)
## Задачи
1. Провести разведочный анализ даных.
2. Сделать кластеризацию клиентов.
3. Обучить модель для скринингового анализа и оценить их.
## Данные
Файл с данными: Тест кандидата на вакансию в Отдел скоринговых систем.xls

## Используемые библиотеки 
pandas, matplotlib.pyplot, numpy, seaborn, sklearn, imblearn, catboost, xgboost.
